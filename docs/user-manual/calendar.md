# Calendar and Reminders

## Calendar View

The Calendar shows all scheduled, active, and completed engagements as date-range blocks.

- **Monthly view**: Full month grid with engagement dots on active days.
- **Weekly view**: Focused 7-day view for detailed planning.
- Toggle between views with the **Month / Week** buttons.

## Engagement Blocks

Each engagement spans from `start_date` to `end_date`. Click any block to open the engagement detail.

## Reminders

Reminders are calculated dynamically on app launch and refreshed every hour:

- **Engagement start**: default 3 days before
- **Engagement end**: default 1 day before
- **Overdue invoices**: from the Invoices page
- **Post-engagement follow-ups**: from Email settings offsets

Dismiss any reminder to hide it permanently.

## Dashboard Widget

The Dashboard shows an **Upcoming** widget with the next 5 reminders sorted by urgency.
