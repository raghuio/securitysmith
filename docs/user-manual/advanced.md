# Advanced Features

## Activity Log

Every mutation in SecuritySmith is recorded in the immutable Activity Log.

- Go to **Activity Log** from the sidebar.
- Filter by: date range, entity type, action type, free-text search.
- Export filtered results to CSV.
- View the full history of any single entity.

## Auto-prune

In **Settings → Advanced**, you can configure automatic pruning of audit logs older than N years. Disabled by default.

## CSV Export

From any list view (Clients, Engagements, Findings), you can export the filtered data to CSV for external analysis.

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | Open command palette / global search |
| `Ctrl+;` | Open AI minibuffer |
| `?` | Show keyboard shortcut help |
| `Esc` | Close drawers and modals |

## Customization

- **Theme**: light, dark, or system
- **Accent color**: choose from palette or enter custom hex
- **Dashboard widgets**: show/hide individual stat cards
- **Sidebar navigation**: hide features you do not use
