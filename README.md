# SecuritySmith

SecuritySmith is a privacy-first, open-source desktop application for security consultants and penetration testers to manage consulting engagements end to end — from initial client contact and scope definition to findings collection, report generation, and retest tracking. All data stays encrypted on your local machine. No servers. No hosting. AI assistance via local Ollama.

> ⚠️ This is a **Work in Progress** project. Core features are being built incrementally and the codebase is not yet ready for production use.
